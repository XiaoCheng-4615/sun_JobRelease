 

# sun_JobRelease 接單系統

### 功能
1. **自定義通知**: 在配置文件中設置通知功能，例如 `config_function`
2. **玩家辨識**: 實現檢查玩家 ID 是否存在並確認是否為自己 ID 的功能
3. **自動標點玩家**: 當有接單玩家時，自動進行標點
4. **通知使用 pNotify**: 預設通知使用 pNotify 
5. **Discord WebHook**:Discord WebHook，以便在特定事件發生時發送通知到 Discord。
